# 🌸 Telegram-бот «Магазин цветов»

Полнофункциональный mini-app для Telegram на **aiogram 3.x** с каталогом, корзиной, скидками, оформлением заказов и админ-панелью.

## Быстрый старт

```bash
cd flower-shop-bot
pip install -r requirements.txt
copy .env.example .env   # Windows
```

Заполните `.env`:
- `BOT_TOKEN` — токен от [@BotFather](https://t.me/BotFather)
- `ADMIN_ID` — ваш Telegram ID ([@userinfobot](https://t.me/userinfobot))

```bash
python app.py
```

## Возможности

### Для пользователя
- **🌷 Каталог** — карточки с фото, описанием, ценой и скидками (старая цена перечёркнута, процент справа)
- **🛒 Корзина** — оформленный вид, +/- количество, удаление позиций, очистка
- **Оформление заказа** — FSM: имя → телефон → адрес → подтверждение
- **📖 История заказов** — статусы с уведомлениями об изменениях
- **👤 Профиль** — сохранённые контактные данные

### Для администратора (`/admin`)
- **Товары:** просмотр, добавление, **редактирование всех полей**, удаление, активация/деактивация
- **🏷 Скидки:** отдельная кнопка, пресеты 5–30% или свой процент
- **Заказы:** просмотр, смена статуса (клиент получает уведомление), удаление
- **🔔 Уведомления** о каждом новом заказе

```

├── config.py

├── cart_storage.py
```

## Технологии

- aiogram 3.x, aiosqlite, python-dotenv
- MemoryStorage (FSM) + in-memory корзина
- SQLite (`flower_shop.db`, создаётся автоматически)
